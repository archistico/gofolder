# GoFolder

## Install
* git clone https://github.com/archistico/gofolder  
* cd gofolder  
  
* Edit your file "list.txt" (use 4 spaces or tab - one folder for line)  
ex.  
First  
Second  
[Tab] Third  

* go build gofolder.go  
* gofolder -h
* gofolder -show
* gofolder -make
* gofolder -file=test.txt
